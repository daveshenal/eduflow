# AZURE AI RAG System

**AI-powered retrieval and generation for micro-education in healthcare compliance**

## Key Components
1. **Data Pipeline**
2. **Retrieval System**
3. **Generation System**
   - General Chatbot
   - Creates micro-learning modules
   - Supports multiple output formats (PDF+voice, assessments)
